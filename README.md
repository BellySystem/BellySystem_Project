# BellySystem — Project Description

BellySystem is an interactive musical interface and performance system developed as an exploration of embodied interaction, gesture recognition, and real-time sound control through body movement. The project investigates how dance gestures and spatial presence can be translated into expressive audiovisual responses using embedded sensing technologies and machine learning.

The system integrates distributed sensors, microcontroller-based data acquisition, and real-time classification models to interpret performer actions. Spatial detection modules allow the identification of presence and movement across a defined interaction area, while inertial sensing enables gesture analysis and dynamic control parameters.

Data acquisition and embedded control were developed within the Arduino ecosystem using ESP32 microcontrollers. Machine learning workflows, including dataset preparation and classifier training, were implemented using Google Colaboratory notebooks. Interactive prototyping, communication, and audiovisual mapping were explored through Max/MSP environments developed by Cycling '74.

BellySystem explores low-latency interaction strategies by combining embedded audio playback, sensor fusion, and gesture classification, aiming to reduce dependency on external computers while maintaining expressive performance capabilities.

The project combines artistic research, interactive system design, and technical development within the field of new musical interfaces and embodied performance technologies.
