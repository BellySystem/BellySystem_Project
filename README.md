# BellySystem

Gesture-driven musical interface for belly dance performance and improvisation.

Embodied interaction · Machine Learning · Real-time Sound Control

## Project Description

BellySystem is an interactive musical interface and performance system developed as an exploration of embodied interaction, gesture recognition, and real-time sound control through body movement. The project investigates how belly dance movements can be translated into expressive audio responses using embedded sensing technologies and machine learning, expanding individual creative agency through embodied musical interaction.

The project explores an inverted relationship between dance and music, where movement becomes the primary source of musical generation rather than a response to pre-existing sound.

The system integrates an inertial measurement unit (IMU), microcontroller-based data acquisition, and real-time classification models to interpret performer actions. Inertial sensing enables gesture analysis and dynamic expressive control through body movement.

BellySystem is designed to support improvisation, personal practice, learning, and exploratory performance scenarios, providing responsive sonic feedback directly linked to body movement.

Data acquisition and embedded control were developed within the Arduino ecosystem using an ESP32 microcontroller. Machine learning workflows, including dataset preparation and classifier training, were implemented using Google Colaboratory notebooks. Interactive audio mapping prototypes were explored through Max/MSP environments developed by Cycling '74.

BellySystem explores low-latency interaction strategies through sensor fusion and gesture classification, aiming to support autonomous musical exploration, improvisation, learning, and performance contexts without requiring accompanying musicians, while maintaining expressive interaction.

The project combines artistic research, interactive system design, and technical development within the field of new musical interfaces and embodied performance technologies.
